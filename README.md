# AXIS

Axis is a dual wheeled balancing robot using a PID controll algorithm.

The robot is controllable with an xbox controller, measures its angle using an IMU sensor and uses 2 DC motors to actuate its wheels.
Encoders are used to increase positioning possibilities for the DC motors.
A custom PCB is used to connect up all components.
